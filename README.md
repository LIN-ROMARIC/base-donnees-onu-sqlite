# SAE1.04 – Création d’une base de données SQLite

## Description
Projet réalisé dans le cadre de la SAE1.04 du BUT Informatique.
Création d’une base de données relationnelle à partir de données démographiques issues de l’ONU.

## Contenu du projet
- Modélisation du schéma relationnel
- Création des tables
- Importation des données
- Écriture de requêtes SQL
- Création de vues pour l’analyse des données

## Technologies utilisées
- SQLite
- SQL
- Outils : DBeaver / SQLiteStudio

## Exemples de requêtes
- Population par pays
- Population par région
- Agrégations (sommes, moyennes)
- Jointures entre plusieurs tables

## Travail en groupe
Projet réalisé en groupe dans un cadre universitaire.  
Ma contribution : participation à la conception de la base et à l’écriture des requêtes SQL.

## Compétences développées
- Modélisation de base de données
- Requêtes SQL
- Analyse de données
- Travail collaboratif
